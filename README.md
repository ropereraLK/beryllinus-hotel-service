# Beryllinus Backend

Backend services for the Beryllinus hotel booking platform.

## Technology Stack

* Java 21
* Spring Boot
* Maven
* Postgres 

## Related Repositories

### Frontend (UI)

The frontend application is available at:

https://github.com/ropereraLK/beryllinus

For frontend setup and usage instructions, refer to the frontend README:

https://github.com/ropereraLK/beryllinus/blob/main/README.md

## Getting Started

```bash
./mvnw spring-boot:run
```

Or build and run:

```bash
./mvnw clean package
java -jar target/beryllinus-backend.jar
```
